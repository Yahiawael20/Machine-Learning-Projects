# Machine Learning Projects

This repository contains a collection of Machine Learning projects implemented using Python and Scikit-learn. The goal is to practice and apply different ML techniques on real datasets.

## 📌 Projects Included

### 📊 1. Regression Project
- Predicting continuous values (e.g. Laptop Price Prediction)
- Data preprocessing and feature engineering
- Model training and evaluation

### 🧠 2. Classification Project
- Loan Status Classification (Predicting loan approval status)
- Supervised learning models
- Accuracy evaluation and metrics

### 📉 3. Clustering Project
- Credit Card Customer Segmentation (Grouping customers based on spending behavior)
- Unsupervised learning
- K-Means and other clustering techniques

## 🤖 Machine Learning Models Used
- Linear Regression
- Ridge / Lasso Regression
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- K-Means Clustering

## 🛠️ Tech Stack
- Python 🐍
- Pandas & NumPy
- Scikit-learn
- Streamlit (for deployment)
- Matplotlib / Seaborn
